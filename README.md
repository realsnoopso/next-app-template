# next-template

A Next.js 13 template for building apps with Radix UI and Tailwind CSS.

## Prerequisites

- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/)

## Setup

### Installation

```bash
# use nvm
$ nvm use

# Install dependencies
$ npm install
```

### Running the app

```bash
# development mode
$ npm run dev

# production mode
$ npm start
```
